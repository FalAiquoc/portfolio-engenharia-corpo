# Engenharia do Corpo — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
A Engenharia do Corpo é uma das maiores franquias de academias do Brasil, com unidade no bairro de Lagoa Nova em Natal/RN. Oferece infraestrutura de ponta, ampla variedade de aulas coletivas e planos com valores populares acessíveis.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout de landing pages de alta conversão.
- https://github.com/tailwindlabs/tailwindui-templates — Padrões e tokens de design para projetos reais.
